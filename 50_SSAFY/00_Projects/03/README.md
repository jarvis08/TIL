# BOXOFFICE 페이지 꾸미기

- 지정하여 속성 값 부여하기
  - tag 별
    - `header{}`
    - `nav{}`
  - 자손 별
    - `.nav-items > li {}`
  - id 별
    - `#section-title {}`
  - 마우스 올렸을 때
    - `.nav-items > li > a:hover {}`
- 사용한 속성 값
  - `color`
  - `float`
  - `text-decoration`
  - `background-image: url()`
  - `text-align: center`
  - `height/width`
  - `line-height`
  - `font-size`
  - `position: absolute/relative/fixed`
  - `display: inline-block`
  - `padding`
  - `top/bottom`
  - `list-style-type: none`
    - `list-style: none`
