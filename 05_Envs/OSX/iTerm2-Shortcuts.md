# iTerm2 Shortcuts

## Terminal 공용

- `Ctrl + A` : 맨 앞으로 삽입점 이동
- `Ctrl + E` : 맨 뒤로 삽입점 이동

<br>

<br>

## Only iTerm

- `Ctrl + w`: 앞의 하나 단어 삭제
- `Cmd + d` : 가로 분할
- `Cmd + shift + d` : 세로 분할
- `Ctrl + f > 탐색 단어 입력 > tab` : 다음 단어 까지 search에 포함
- `Ctrl + f > 탐색 단어 입력 > Opt + return` : 해당 단어 복사하여 command line에 입력
- `Cmd + return` : toggle full screen
- `Cmd + k` : clear
- `Cmd + Shift + ]` : next tab
- `Cmd + Shift + [` : previous tab

### Custom

- `Opt + w` : next buffer
- `Opt + Shift + w` : previous buffer
