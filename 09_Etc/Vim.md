# Vim

Vi의 advanced 버전인 Vim의 단축키입니다.

<br>

### 기본 상태

`/단어`: 단어 검색

`dd`: 행 삭제

`u`: 되돌리기(=`cmd+z` || `ctrl+z`)

<br>

### Insert Mode

`i`: Insert Mode 실행

<br>

### Visual Mode

`v`: Visual Mode 실행 및 케릭터 단위 선택

`V`: 줄(행) 단위 선택

`ctrl+v`: 블록으로 선택(여러 줄, 공통 범위 선택에 용이)

`d`: 잘라내기

`y`: (Yank) 복사하기

`p`: (Put)붙여넣기
