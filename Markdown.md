## CLI(Command Line Interface)

: 유닉스 shell(Linux, Git Bash, ...)

: CMD, Powershell

`ls` : list 목록

`cd`: 지정 위치로 이동

`pwd`: 현위치 (point working directory)

`mkdir`: 폴더 생성

`code .`: vs code 현위치에서 실행

`touch 문서.txt`: 문서를 생성

`rm 문서.txt`: 문서 삭제

## Markdown Language

`- + Enter`: black point

`(```python + Enter)`: 블록 코드 강조 (back tick)

```
(Ctrl + Enter) : 빠져나가기
```

`--- + Enter`: 줄긋기

`Ctrl + T`: 표 만들기

`back tick + 내용 + back tick`: 코드블록에 내용 넣기

`> ` 단 띄우기
