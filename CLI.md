## Unix Shell

`sudo + 명령어` : 관리자 권한으로 명령어 실행

`ls` : 현재 디렉토리 목록

`ls -al` : 숨김파일까지 목록

`cd`: 지정 위치로 이동

`pwd`: 현위치 (point working directory)

`mkdir`: 폴더 생성

`code .`: vs code 현위치에서 실행

`touch 문서.txt`: 문서를 생성

`rm 문서.txt`: 문서 삭제

`mv 현재명 변경명` : 파일명 변경, 위치 이동

`.` : 현위치

`./` : 현위치에서 부터 path 설명

`~/` : home으로부터 path 설명

`chmod` : 권한 부여

tar

`echo` : cli의 print

`echo $PATH` : path 보여줌
