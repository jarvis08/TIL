# System Memory

## Memory Structure

### Code

- Operating System의 설정을 저장하는 공간
- C 와 같은 언어의 경우 조작 가능

### Data

- Built-in 함수
- Global Variable

### Stack

- Local Variable

- 사용자 정의 함수

- 함수 호출 저장

  재귀 함수 사용 시 함수 호출이 System Memory의 Stack에 누적되며,

  Stack Memory를 넘어서는 함수 호출을 제한

### Heap

- 포인터
- 동적 할당
