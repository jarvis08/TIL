## Box Model

## 모든 HTML은 네모로 구성

- 원은 네모를 돌려깎아 사용

  `Margin` > `Border` > `Padding` > `Content`

<br>

### 1. Margin

테두리 바깥의 외부 여백

배경색 지정 가능

<p> <h1~> 는 기본적으로 margine이 있어, 줄 띄우기가 가능

<br>

### 2. Border

- 테두리 영역

  `padding`과 `margin`의 경계를 테두리로 설정

  `solid` 등의 모양 설정 가능

- MDN Docs

  https://developer.mozilla.org/ko/docs/Web/CSS/border-style

<br>

### 3. Padding

테두리(border) 안쪽의 내부 여백

요소에 적용된 배경의 컬러/이미지는 패딩까지 적용

<br>

### 4. Content
